# 현재 파이썬 환경 패키지 정리

## 확인 결과
- 작업 공간에서 별도의 `venv` 또는 `pyvenv.cfg`는 확인되지 않았습니다.
- `python` 명령은 없었고, `python3`만 사용 가능했습니다.
- `python3`에는 `pip`가 없어 `pip list` 기반 목록을 직접 추출할 수 없었습니다.
- 대신 현재 `python3` 환경에서 import 가능한 모듈 목록을 정리했습니다.

## 환경 정보
- Python 버전: `3.12.3`
- 감지된 import 가능 모듈 수: `327`

## 참고
아래 목록에는 다음이 함께 포함될 수 있습니다.
- 표준 라이브러리 모듈
- 시스템 전역 설치 패키지
- 서드파티 패키지
- 일부 내부/배포판 전용 모듈

정확한 `pip` 설치 패키지 목록이 필요하면, 실제 venv를 활성화한 뒤 `pip list`를 실행할 수 있어야 합니다.
