# 인사/급여/복리/지방공기업법 질의 간단 보고서 스킬

## 사용 시기
- 사용자가 인사, 급여, 복리 규정 또는 지방공기업법 관련 질문을 할 때
- 사용자가 규정 비교(인사 vs 급여 vs 복리 vs 법령)를 요청할 때
- 사용자가 "간단히 보고서 형태로 정리"를 요청할 때

## 목적
- PDF 규정 문서를 근거로 답변하고, 실무자가 바로 활용 가능한 간단 보고서 형태로 정리한다.
- RAG + MCP + Skill 통합 방식(01-hybrid-agent.ipynb)의 운영 원칙을 따른다.

## 도구 사용 우선순위
1. 파일 상태 점검형 질문:
- `list_regulation_sources`

2. 일반 규정 검색형 질문:
- 1차: `search_regulations(query, category="all")`
- 2차: 필요 시 `get_regulation_page(category, page)`

3. 카테고리 지정 질문(인사/급여/복리/법령):
- 1차: `search_regulations(query, category="personnel|salary|welfare|law")`
- 2차: `get_regulation_page(category, page)`

4. 규정 비교 질문:
- 1차: `compare_regulation_topic(topic)`
- 2차: 필요 시 `search_regulations`로 근거 보강

5. 계산 포함 질문:
- 덧셈: `add`
- 곱셈: `multiply`

6. 규정 외 일반 지식 보강이 필요한 경우:
- `retrieve`

## 보고서 양식 (반드시 아래 형식 사용)

### [인사/급여/복리/지방공기업법 질의 간단 보고서]
- **질문 요약**: (사용자 질의 핵심 1~2문장)
- **질의 유형**: (검색형 / 비교형 / 검증형 / 계산형)

#### 1. 비교정리
- 인사/급여/복리/법령 관점의 핵심 차이와 결론을 2~4개 불릿으로 제시

#### 2. 실무적용 체크리스트
- 담당자가 즉시 실행 가능한 액션 2~5개

#### 3. 주의사항
- 적용 범위의 예외
- 절차/승인 체계의 유의점
- 규정 문구 불명확/미검색 시 추가 확인 포인트

#### 4. 근거 안내
- 근거 문서 내용은 우측 사이드 패널 Preview를 참고하도록 1~2문장으로 안내

## 작성 규칙
- 추정으로 단정하지 않는다.
- 도구로 확인한 근거만 사용한다.
- 가능하면 인사/급여/복리/법령 관점을 분리해서 설명한다.
- 본문에 source/page를 나열하지 않고, 우측 Preview를 근거 참조로 사용한다.
- 답변은 간결한 한국어로 작성한다.

## 실패/예외 처리
- 결과 없음: 검색어 축약 또는 카테고리 변경 후 재검색
- 페이지 미존재: 해당 사실을 명시하고 다른 근거 제시
- 파일 누락(MISSING): 파일 상태를 먼저 보고하고 답변 범위를 제한
